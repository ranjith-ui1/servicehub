# ServiceHub - Frontend

ServiceHub is a service provider management and booking platform frontend built using React.js.

The platform connects users with multiple service providers such as electricians, plumbers, carpenters, painters, cleaners, AC repair technicians, tutors, and other professionals.

The application supports three different user roles:

- User
- Service Provider
- Admin

Each role has its own dashboard and functionality.

## 📌 Project Overview

ServiceHub provides a single platform where users can search for professional service providers based on:

- Service category
- Provider name
- City
- Rating
- Experience
- Availability
- Price

Users can view service provider details, add services to favorites, and manage their bookings.

Service providers can manage their profiles, services, service requests, and bookings.

Administrators can manage users, service providers, services, and bookings.

---

## 👥 User Roles

### 1. User

Users can:

- Register an account
- Login to the platform
- Browse service providers
- Search for services
- View service details
- Book services
- Manage bookings
- Add services to favorites
- Manage their profile

---

### 2. Service Provider

Service providers can:

- Register as a service provider
- Login to their provider dashboard
- Manage their profile
- Add and manage services
- View service requests
- Accept or reject requests
- Manage bookings

Example professions:

- Electrician
- Plumber
- Carpenter
- Painter
- Cleaner
- Mechanic
- AC Repair Technician
- Tutor
- Gardener
- Computer Repair Technician

Multiple service providers can offer the same type of service.
